# Advanced EDA & Feature Engineering

A step-by-step Python project covering:
- Missing data imputation (Mean, Median, KNN)
- Outlier detection and handling (Z-Score, IQR)
- Feature engineering (6 new predictive features)

**Tools:** Python, Pandas, NumPy, Scikit-learn, SciPy, Matplotlib

## Project Structure
- `src/` — Python scripts, one per pipeline step
- `data/` — raw input data
- `outputs/` — cleaned, ML-ready dataset
